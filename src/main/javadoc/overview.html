This Java library implements Zhang's camera calibration algorithm:

Reference: Z. Zhang, "A flexible new technique for camera calibration", IEEE Transactions on Pacodeern Analysis and Machine Intelligence, 22(11), pp. 1330-1334 (2000).

Note that this implementation performs the geometric part of camera calculation but does not provide interest point detection and point correspondence calculations. See this extensive report for details.

Author: Wilhelm Burger

Dependencies: Requires ImageJ, the ImagingBook Common library and Apache Commons Math.

Installation:

  1. Place imagingbook-common.jar and imagingbook-calibrate.jar in ImageJ/jars/.
  2. Place the sample plugins in ImageJ/plugins/.
  3. Restart ImageJ.
